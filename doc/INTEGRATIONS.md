# Integrations

[Back to README](../README.md)

## Purpose

Documents external systems, endpoint details, payloads, retries, rate limits, batching, and mappings.

## Current integrations

### Neon Postgres

The app connects to Neon through `DATABASE_URL` using `@neondatabase/serverless` for server-side SQL queries.

### Vercel

The app is designed to run on Vercel as a Next.js application.

## Not implemented

ERP/MES integrations, barcode systems, machine integrations, and exports are future phase items.
